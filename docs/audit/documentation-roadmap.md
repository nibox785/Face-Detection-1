# Documentation Roadmap

Tài liệu này là lộ trình tạm thời để cải thiện tài liệu dự án Face Detection.

## Mục tiêu ngắn hạn

- Hoàn thiện các tài liệu audit ở mức tổng quan và có cấu trúc rõ ràng.
- Phân biệt rõ giữa runtime, training, evaluation và testing.
- Bổ sung tài liệu cho các endpoint API và luồng frontend.

## Mục tiêu trung hạn

- Chuẩn hóa hướng dẫn cài đặt và chạy dự án.
- Thêm tài liệu về cấu trúc thư mục và quy trình phát triển.
- Tạo tài liệu cho từng module chính: backend, models, quality, segmentation.

## Mục tiêu dài hạn

- Xây dựng tài liệu chính thức cho onboarding mới.
- Có một quy trình cập nhật docs cùng với thay đổi code.
- Bổ sung mô tả về benchmark, metrics và deployment.

## Gợi ý ưu tiên

1. Tài liệu API
2. Tài liệu cấu trúc project
3. Tài liệu training và export model
4. Tài liệu testing và CI
5. Tài liệu deployment

> Đây là tài liệu tạm thời, không phải tài liệu chính thức.
