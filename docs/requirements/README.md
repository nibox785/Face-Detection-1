# Requirements (Reverse Engineered)

Tài liệu này là bản reverse engineer requirements từ bằng chứng triển khai hiện có trong repository. Nội dung được rút ra từ các file mã nguồn và cấu trúc hiện tại, không bổ sung tính năng mới.

## Nội dung

- [functional.md](functional.md): yêu cầu chức năng
- [non-functional.md](non-functional.md): yêu cầu phi chức năng
- [constraints.md](constraints.md): ràng buộc
- [assumptions.md](assumptions.md): giả định
- [acceptance.md](acceptance.md): tiêu chí chấp nhận
