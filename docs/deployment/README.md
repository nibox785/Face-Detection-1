# Tài Liệu Triển Khai Hệ Thống

Thư mục này chứa các tài liệu hướng dẫn cấu hình, cài đặt và chạy ứng dụng AI Face Detection ở các môi trường khác nhau.

## 📌 Các Tài Liệu Hướng Dẫn

1.  **[Cài đặt Local (local.md)](local.md)**: Hướng dẫn cài đặt Python, thư viện, cấu hình số luồng tính toán CPU và cài đặt GPU tăng tốc.
2.  **[Triển khai Docker (docker.md)](docker.md)**: Kịch bản build ảnh và chạy container hóa.
3.  **[Môi trường Production (production.md)](production.md)**: Hướng dẫn tối ưu hóa bảo mật, CORS, Reverse Proxy (Nginx) phục vụ thực tế.
