# Cấu trúc phân tầng tài liệu

## Mục đích

Tài liệu này mô tả mối quan hệ giữa tài liệu nền tảng, tài liệu chuyên môn và tài liệu hỗ trợ.

## Phân tầng

- Tài liệu nền tảng xác định mô hình tài liệu tổng thể, tiêu chuẩn và kỳ vọng về quản trị.
- Tài liệu chuyên môn mô tả các lĩnh vực cụ thể và cung cấp nội dung tham khảo chính cho các lĩnh vực đó.
- Tài liệu hỗ trợ cung cấp bối cảnh, ví dụ, ghi chú kiểm tra hoặc hướng dẫn bổ sung.

## Quy tắc liên hệ

- Tài liệu nền tảng thiết lập cách các tài liệu khác nên được tạo và duy trì.
- Tài liệu chuyên môn nên tuân theo cấu trúc và thuật ngữ giống như lớp nền tảng.
- Tài liệu hỗ trợ nên vẫn phù hợp với nguồn sự thật chính cho chủ đề mà chúng hỗ trợ.
